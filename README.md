# Privacy & Mobility Data Simulator

A privacy-preserving mobility data platform that generates realistic synthetic movement datasets from OpenStreetMap data and evaluates anonymization techniques through an interactive challenge platform.

The project simulates daily mobility patterns of virtual users (home, work, and routine activities), generates synthetic trajectories using graph-based routing algorithms, and provides tools to assess privacy protection and re-identification risks.

## Key Features

* Synthetic mobility data generation from real-world maps
* Agent-based simulation of daily movement patterns
* Privacy and anonymization evaluation framework
* Interactive challenge platform for testing privacy-preserving techniques
* Visualization and analysis of mobility datasets
